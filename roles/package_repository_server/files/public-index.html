    Repository Signing Key

    For security, our repository is signed with a GPG key. To properly add this repository to your system, follow these steps:

    1. Create a directory for keyring files (if it doesn't exist)

    2. Download the GPG key

    3. Set proper permissions for the key

    4. Add the repository with reference to the key

    5. Update your package index

    6. Now you can install packages

    The following method uses the deprecated [trusted=yes] option and is not recommended for security reasons. Please use the method above instead.

    1. Add the repository to your sources list (NOT RECOMMENDED)
    2. Update your package index
    3. Install a package
